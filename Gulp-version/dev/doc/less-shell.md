# Components #

## btnMakers

### Usage:
	.btn2{
	  .btn-maker(20px);
	}

### Params: 

* **String** *size* of the padding

* **non-string** *margin* value for the bottom of the button.

* **non-string** *padding* the default padding value around the button.

### Return:

* **none** none
